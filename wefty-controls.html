Controls

Pick where the LASSO and JUMP buttons sit. Saved per-game so Run and Climb can have different layouts.

Wefty Run

Auto-runner — Lasso + Jump on opposite sides by default.

Want exact positions? Tap Custom, start Wefty Run, then long-press LASSO or JUMP to drag them anywhere on the screen. Tap Save in the floating bar when you're done.

Wefty Climb

Vertical climber — same layout system as Run.

Auto-jump assist
Wefty auto-jumps after sitting on a ledge for ~½ sec. You still steer + lasso. Manual JUMP still works.
