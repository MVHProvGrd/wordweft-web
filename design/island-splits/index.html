Wefty Run · Island Depth Splits

Review preview of the per-island back/front layer crops used by the Wefty Run depth renderer. Trees sit behind Wefty (the _back layer); grass, flowers, mushrooms, and the opossum render in front (the _front layer).

Split rule: user-traced red line from run_island_vX (Run path).png defines the walking surface. Pixels above the line with a vertical opaque run ≥ 20% of PNG height stay in _back (trees). Everything else — thin grass blades, mushroom tops, opossum — moves to _front. Ground / flowers below the line are in _front by default. Generated by /tmp/split_island_depth.py.
