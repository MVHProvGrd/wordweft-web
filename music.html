WEFTY-letters power-up

SFX

Wefty Run — fires when you collect all five W-E-F-T-Y letters. wefty-run.html dispatches the 'power-up' event, which plays this mp3 plus a 3-note ascending C5-E5-G5 fanfare via Sound.playGameStart() (the synth jingle layered on top). Same flow on Android via SoundManager.playGameStart().

Originals — preserved for posterity

These are the pre-Lyria 3-overtone synth versions of the same Jazz and Chiptune melodies, rendered offline from the exact algorithm in SoundManager.kt (sine fundamental at 0.7 + sine 2× at overtone2 + sine 3× at overtone3, with a 10% linear attack/release envelope). The live game now plays the Lyria versions above; these stay around in case the originals come back into rotation.
