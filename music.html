Cue grid

29 of these cues now play in-game on both web + Android at the call sites identified by the audit (jumps, letter pickups, toasts, biome transitions, grade fanfares, achievements, screen nav, etc.). Three cues stay on shelf for now: sfx_ledge_grab + sfx_ledge_slip by request, and sfx_hint_reveal until the hint feature ships. Synth-rendered from the toolkit at tools/gen_sfx.py; regenerate or tweak any single cue without touching the others.

Tier 1 — gameplay readability

sfx_word_reject

Invalid word / profanity buzz. Falling 600→380 Hz square-ish, ~160 ms.

sfx_ledge_grab — on shelf

Climb — feet hit ledge. Low thump 85 Hz + brief gravel, ~110 ms.

sfx_ledge_slip — on shelf

Climb — grip fails. Pitch-down whine 700→180 Hz + gravel, ~220 ms.

sfx_biome_change

Run/Climb biome boundary cross. C5-E5-A5 stinger + sparkle, ~400 ms.

sfx_letter_collect_w

Run — first WEFTY letter. Plucky pizzicato C5.

sfx_letter_collect_e

Run — 2nd letter. Pizzicato E5 (one step up from W).

sfx_letter_collect_f

Run — 3rd letter. Pizzicato G5.

sfx_letter_collect_t

Run — 4th letter. Pizzicato B5.

sfx_letter_collect_y

Run — final letter Y; D6 pizzicato + sparkle tail (precedes the existing power-up cue).

sfx_spring_bounce

Run — spring-pad bounce. Boing pitch-rise 380→1100 Hz exp, ~160 ms.
